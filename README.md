# Computer-Network

这是一个关于计算机网络的学习项目，包含了结合实战排错的网络基础知识体系。

## 内容结构

*   **知识库/**: 存放所有的核心学习资料和文档
    *   `全栈大一统学习路线.md`: 涵盖了从网络底层基础到架构设计的完整学习指引。
    *   `排错案例_IP与代理冲突分析.md`: 基于终端实际日志分析 502 Bad Gateway 报错、代理劫持及 CIDR 匹配问题。
    *   `tex_compile/`: 包含了生成的 PDF 讲义及相关的 LaTeX 源码。在这个版本中，使用通俗易懂的“寄快递”比喻重新讲解了艰涩的网络理论。
*   **.agent/**: 智能体辅助开发的上下文和会话记录目录。

## 学习建议

本项目专门针对没有深厚计算机基础的初学者。推荐的学习顺序为：
1. 仔细阅读 PDF 讲义 `知识库/tex_compile/network_theory.pdf`。
2. 结合 `排错案例_IP与代理冲突分析.md` 进行实操验证（如使用 `curl -v` 和 `--noproxy '*'`）。
3. 参考 `全栈大一统学习路线.md` 进入更深一步的源码与架构探究。

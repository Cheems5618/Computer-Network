# .agent 智能体框架

欢迎，新智能体。这个文件夹是你快速理解当前项目的入口。

## 框架结构

```
.agent/
├── AGENT.md          # ← 你在这里（框架说明 + 阅读指引）
├── project.md        # 项目概述（是什么、用什么技术、目录结构）
├── rules.md          # 行为规则（代码风格、禁止事项、必须遵守的流程）
├── context.md        # 当前工作上下文（正在做什么、下一步计划）
├── memory/           # 记忆系统
│   ├── MEMORY.md     #   记忆索引（列出所有记忆条目）
│   ├── decisions.md  #   关键决策 + 原因
│   ├── patterns.md   #   项目中的代码模式/惯例
│   └── lessons.md    #   踩过的坑 / 经验教训
└── sessions/         # 会话记录
    └── YYYY-MM-DD.md #   按日期记录每次会话摘要
```

## 标准工作流程

每次被召唤到本项目时，请按以下顺序阅读：

1. **project.md** — 了解这是什么项目，用什么技术栈，目录怎么组织的
2. **rules.md** — 了解行为约束：什么能做、什么不能做、代码风格要求
3. **context.md** — 了解当前工作状态：正在做什么、下一步要做什么
4. **memory/MEMORY.md** — 扫一眼记忆索引，看有没有和当前任务相关的历史记忆
5. **开始工作**

> 不需要逐字精读，快速扫描抓住关键信息即可。遇到不确定的事情时，回查对应文件。

## 工作结束后

请在 `sessions/` 下创建或更新当天的会话记录，简要记录：
- 做了什么
- 做了哪些关键决策
- 有什么需要下一个智能体注意的事项

如果产生了值得长期保留的记忆（重要决策、新的代码模式、踩坑经验），请更新 `memory/` 中对应的文件，并同步更新 `MEMORY.md` 索引。

## 设计原则

- **纯 Markdown** — 无需任何工具即可阅读
- **快速扫描** — 每个文件控制在 200 行以内
- **渐进式** — 按需深入，不用一次读完所有文件
